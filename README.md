Git príkazy a ich vysvetlenie 

Nastavenie používateľa pre všetky repozitáre

git config --global user.name "Meno"
git config --global user.email "email@example.com"



Naklonuje existujúci repozitár z URL

git clone [url]


Stiahne zmeny zo vzdialeného repozitára a spojí ich

git pull

Stiahne zmeny zo vzdialeného repozitára bez spojenia

git fetch


Pripraví všetky zmenené súbory pre commit

git add .


Vytvorí commit so zmenami a popisom

git commit -m "správa"


Odošle lokálne commity do vzdialeného repozitára

git push
